How I chose to use Bootstrap and Javascript:
I used Bootstrap for containers and grids mainly, but also for aesthetic-looking buttons and header text; also, I thought the color names were easy to access and a nice touch.
I used JavaScript to implement a "Quote of the Day" button in the footer of my page. This simply creates an alert when the button is clicked, and displays a quote selected by me.
